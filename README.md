# kfm-ios-test
Test App for PT Buana Vista Komputama

Made using Xcode & Swift 5

Third Party API used => www.weatherapi.com

Pods included:
- Alamofire for Networking
- Kingfisher for Image URL Processing
- DropDown for dropdown function
